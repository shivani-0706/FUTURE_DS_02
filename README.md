# FUTURE_DS_02

Project Title
Customer Retention & Churn Analysis

Objective
Analyze customer churn patterns and identify retention drivers using the Telco Customer Churn Dataset.

Tools Used
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook

KPI Metrics
Total Customers
Churn Rate
Average Customer Lifetime Value

Analysis Performed
Churn by Contract Type
Churn by Tenure Group
Churn by Payment Method
Churn by Tech Support
CLV by Contract Type
Contract × Tenure Heatmap

Key Insights
• Month-to-month customers have the highest churn.

• Most churn occurs in the first year.

• Electronic Check users are more likely to leave.

• Customers without Tech Support show elevated churn.

• Long-term contracts generate significantly higher CLV.

• Customer retention improves as tenure increases

Business Recommendations

• Promote annual and two-year contracts.

• Improve onboarding during the first 12 months.

• Encourage automatic payment methods.

• Bundle Tech Support with premium plans.

• Launch targeted retention campaigns.

• Reward loyal long-term customers.
